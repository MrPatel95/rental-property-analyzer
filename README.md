# Rental Property Analyzer - Streamlit App

This is a simple web-based rental property analyzer built with Streamlit.  
It allows you to input rental property details and calculate key metrics such as:
- Cap Rate
- Cash on Cash Return
- Monthly Cash Flow
- GRM, and more
